# 快速启动指南

## ⚠️ 重要提示

**必须在项目目录下运行命令！**

项目目录是：`E:\code\peace\github\fast-ai-gallery`

## ✅ 正确的启动步骤

### 步骤1：进入项目目录

在 PowerShell 或 CMD 中执行：

```powershell
cd E:\code\peace\github\fast-ai-gallery
```

### 步骤2：启动服务器

```powershell
npm run dev
```

### 步骤3：等待启动完成

您应该看到类似这样的输出：

```
  ▲ Next.js 16.0.3
  - Local:        http://localhost:3000
  - Ready in 2.3s
```

### 步骤4：访问网站

在浏览器中打开：http://localhost:3000

## 🚀 一键启动（推荐）

### 方法1：使用启动脚本

直接双击 `start-dev.bat` 文件，脚本会自动：
1. 进入正确的目录
2. 检查依赖
3. 启动服务器

### 方法2：在 VS Code 中

1. 在 VS Code 中打开 `fast-ai-gallery` 文件夹
2. 按 `Ctrl + ~` 打开终端
3. 终端会自动在项目目录下
4. 运行 `npm run dev`

## ❌ 常见错误

### 错误：ENOENT: no such file or directory

**原因**：在错误的目录下运行命令

**解决**：
```powershell
# 先进入项目目录
cd E:\code\peace\github\fast-ai-gallery

# 然后再运行
npm run dev
```

### 错误：端口被占用

**解决**：使用其他端口
```powershell
npm run dev -- -p 3001
```

## 📍 验证

启动成功后：
- ✅ 终端显示 "Ready" 消息
- ✅ 浏览器可以访问 http://localhost:3000
- ✅ 看到 AI Gallery 首页和所有 Demo

## 🛑 停止服务器

在运行服务器的终端窗口按 `Ctrl + C`
