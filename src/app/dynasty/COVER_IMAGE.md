# 封面图说明

## 要求
- **文件名**：`cover.png`
- **位置**：`src/app/dynasty/cover.png`
- **建议尺寸**：1200x630px
- **格式**：PNG 或 JPG

## 设计建议
封面图应该体现"中国朝代更迭"的主题，可以包含：
- 中国历史元素（如古代建筑、文字、图案等）
- 时间轴或朝代更迭的视觉元素
- 深色主题，与项目整体风格一致
- 可以包含朝代名称或相关文字

## 创建方式
1. 使用设计工具（Figma、Photoshop、Canva等）
2. 使用AI生成工具（Midjourney、DALL-E等）
3. 使用项目截图
4. 使用在线设计工具

## 临时方案
如果暂时没有封面图，可以使用项目截图或创建一个简单的占位图。
