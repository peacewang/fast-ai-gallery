# 本地部署和运行指南

## ✅ 环境检查

- Node.js 版本：v22.17.1 ✅
- npm 版本：10.9.2 ✅
- 项目构建：已通过 ✅

## 🚀 启动方法

### 方法1：使用启动脚本（推荐）

1. 双击运行 `start-dev.bat` 文件
2. 等待服务器启动完成
3. 在浏览器中访问 http://localhost:3000

### 方法2：使用命令行

1. 打开 PowerShell 或 CMD
2. 进入项目目录：
   ```powershell
   cd E:\code\peace\github\fast-ai-gallery
   ```
3. 启动开发服务器：
   ```powershell
   npm run dev
   ```
4. 等待看到以下输出：
   ```
   ▲ Next.js 16.0.3
   - Local:        http://localhost:3000
   - Ready in X.Xs
   ```
5. 在浏览器中访问 http://localhost:3000

### 方法3：使用 VS Code 终端

1. 在 VS Code 中打开项目
2. 按 `Ctrl + ~` 打开终端
3. 运行 `npm run dev`
4. 点击终端中显示的链接，或手动访问 http://localhost:3000

## ⚠️ 常见问题

### 问题1：端口 3000 被占用

**解决方案**：使用其他端口
```powershell
npm run dev -- -p 3001
```
然后访问 http://localhost:3001

### 问题2：ERR_CONNECTION_REFUSED

**可能原因**：
1. 服务器未成功启动
2. 防火墙阻止连接
3. 端口被其他程序占用

**解决步骤**：
1. 检查终端是否有错误信息
2. 确认服务器已完全启动（看到 "Ready" 消息）
3. 尝试使用 127.0.0.1:3000 代替 localhost:3000
4. 检查防火墙设置

### 问题3：依赖未安装

**解决方案**：
```powershell
npm install
```

### 问题4：编译错误

**解决方案**：
```powershell
# 检查类型错误
npm run typecheck

# 清理并重新构建
npm run build
```
